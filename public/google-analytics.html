Configuration SEO pour Worldlink Logistics Mauritius

1. Google Search Console

Pour configurer Google Search Console :

  1. Allez sur Google Search Console
  2. Ajoutez votre propriété : https://worldlink.mu
  3. Choisissez la méthode de vérification par fichier HTML
  4. Téléchargez le fichier de vérification fourni par Google
  5. Placez-le dans le dossier public/ de votre projet

2. Google Analytics 4

Pour configurer Google Analytics :

  1. Allez sur Google Analytics
  2. Créez une nouvelle propriété pour worldlink.mu
  3. Obtenez votre ID de mesure (format : G-XXXXXXXXXX)
  4. Ajoutez le code de suivi dans votre composant SEO
// Exemple de code Google Analytics à ajouter dans le composant SEO <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-XXXXXXXXXX'); </script>
